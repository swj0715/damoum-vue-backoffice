## 개요
- 다모음 애플리케이션 백오피스 Frontend 입니다. 
- Vuetify (Material UI) 기반으로 개발 되어있습니다. 
  - 컴포넌트들은 [Vuetify 공식 홈페이지](https://vuetifyjs.com/en/) 를 참조하세요.

## 실행 & 빌드 방법
### Build
원하는 profile을 build 명령에 지정 해 줍니다. 아래는 dev profile 빌드 예시입니다.
```shell
yarn build:dev
```
### run
```shell
yarn dev
```
